Лабораторная работа №3
Вариант 10: ранговый фильтр, маска - косой крест, ранг 4/5

Состав архива:
- src/main.py - основной код решения;
- src/requirements.txt - зависимости;
- input/ - входные изображения;
- output/grayscale/ - результаты для полутоновых изображений;
- output/binary/ - результаты для монохромных изображений;
- output/contact_sheets/ - сводные листы "исходное / фильтр / разность";
- output/summary_metrics.csv - таблица с числовыми метриками;
- report/Отчет_ЛР3_Вариант_10.docx - отчет в формате Word;
- task/Лаба 3. Фильтры и морфология.pdf - исходное задание.

Как запустить повторно:
python src/main.py --input_dir ./input --output_dir .
